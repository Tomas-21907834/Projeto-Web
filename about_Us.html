About Us

Aqui falamos sobre este Website

Esta é a página fala acerca de todas as técnicas usadas neste Website através de tabelas e videos. Mostramos tambem um Mapa do Website com expansão completa do menu e todos submenus e tambem uma imagem com os layout responsivos para pc e mobile, e por fim um video a mostrar todos os detalhes e técnicas usadas por nós.

Técnicas usadas

Carateristicas Elementos/Técnicas Exemplos
Layouts Usou-se o CSS grid e CSS flexbox para construção de layouts Usou-se CSS grid: a fazer o layout da página principal e tambem para posicionar as imagens nas novidades.
Usou-se CSS flex: no menu navegação e para posicionar imagem de fundo na sec principal do home e para fazer o formulário.
Responsividade Foram usadas media queries que adaptam o layout para telemovel e PC consoante o tamanho da janela do browser Usamos os media queries no display grid para modificar o nosso layout consoante o tamanho da página para que o site se torne responsivo.
Seletores Foram usados seletores Universal, tipo, classe, id, pseudo-classe, pseudo-elemento Estes selectores serviram para mudar o tipo de letra (universal), para mudar as cores ou backgrounds (tipo), para modificar através do css coisas no site(classe), para modificar coisas unicas no site(id), e para embelezar utilizando a propriedade hover(pseudo-classe).
Etiquetas HTML5 main, header, body, head, nav, article, footer, figure, section, figure, figcaption, img, table, from, fieldset, iframe, legend. Usamos o HTML5 main, header, body, head, nav, figure, section, footer em todas as páginas.
Usamos article figure, figcaption, img em páginas com imagens
Usamos table na página About Us.
Usamos fieldset, iframe, legend para o Formulário.
Animações Foram feitas animações usando keyframes para a página da galeria A primeira animação faz com que 2 imagens estejam sempre a abanar e a outra animação faz com que as bordas das imagens estejam a mudar de cor constantemente.
